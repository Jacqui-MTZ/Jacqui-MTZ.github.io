SDK PARA WEB

SDK es el acrónimo de “Software Development Kit” (Kit de desarrollo de software). El SDK reúne un grupo de herramientas que permiten la programación de aplicaciones móviles. Este conjunto de herramientas se puede dividir en 3 categorías:

  • SDK para entornos de programación o sistemas operativos (iOS, Android, etc.)
  • SDK para el mantenimiento de aplicaciones
  • SDK de marketing y publicidad
