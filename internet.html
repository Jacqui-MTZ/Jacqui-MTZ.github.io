Internet viaja atravez de flujos digitales que son identificados por 010101.

¿QUE ES DIRECCION IP?

Este protocolo es un conjunto de reglas para la comunicación a través de Internet, ya sea el envío de correo electrónico, la transmisión de vídeo o la conexión a un sitio web. Una dirección IP identifica una red o dispositivo en Internet.

Una Direcion IP es una serie de numeros por las cuales se identifican las paginas web. Para las personas es muy dificil recordar las Dirreciones IP de las paginas web, para eso se utilizan los nombres de dominio.

por ejemplo;

  • Amazon.com
  • Yotube.com
